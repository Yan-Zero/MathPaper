# MathPaper

数学试卷，Latex 排版，多数的来源是网络。

本库所采用的是单页 A4 的格式。因为这样子兼容性最强，可以轻松改成 A3。具体参考高考试卷LaTeX模板中的2017quanguo3.tex

自定义宏包在 Package 文件夹下，记得安装。
